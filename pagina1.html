Bienvenidos

Este es el Objeto Virtual de Información O.V.I. del curso de Diseños de Sitios Web en el cual encontrara información acerca de algunas propiedades fundamentales del modelo de caja de CSS3.

La información de este OVI es fundamental para poder continuar el aprendizaje de las propiedades de estilos que nos trae CSS3 ya que contiene documentación que le ayudara a comprender los conceptos básicos y ejemplos para un mejor entendimiento. Entre la gran cantidad de propiedades que trae CSS3 se han seleccionado las siguientes:

• Display

• Position

• Float

• Overflow
